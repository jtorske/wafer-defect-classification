# Wafer Defect Pattern Classification

This project explores automated classification of semiconductor wafer defect patterns
using deep learning models.

## Models

- ResNet-18 (trained from scratch)
- Vision Transformer (ViT-Tiny, fine-tuned)

## Key Focus Areas

- Handling severe class imbalance
- Comparing CNNs vs Transformers
- Per-class evaluation with confusion matrices
